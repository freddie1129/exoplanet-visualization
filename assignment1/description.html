Describe the Dataset

  1. What is the subject matter of the dataset (what is it about?)

    2. This dataset is a collection of 3,573 exoplanets found in the past decades. Every row in this dataset represents an exoplanet with 25 main attributes, like the mass, identifier, discovery year, radius, etc. This dataset might be updated later as the new exoplanets be discovered, and the visualization of this dataset can benefit to analyze the natural of exoplanets and find the general features of exoplanets.

  2. What is the origin of the dataset -- how do you think it was generated or compiled?

    3. The dataset is generated from the astronomers’ research results. Someone collected the data of exoplanets found by astronomers and created this dataset. Then a new record is added to the dataset once there is a new exoplanet to be found. The dataset is relatively static in this course assignment, while it would be changed in the real world as the new exoplanets been found.

  3. Is any cleanup needed before it is usable in a visualisation? How would you perform the cleanup?

    4. Yes. Some values of attributes do not exist in some rows, which are not suitable for visualisation. I plan to find out these items and delete or provide a default value for them according to the different situation as a clean-up of this dataset.

    5. Whether the data need to be pre-processed depends on the different type of the visualization job. For this assignment, the data should be pre-processed in some scenario. For example, a statistic processing would be very necessary for drawing a bar chart representing the amount the exoplanets been found in years. While in other cases, like the plot of radium for a list of exoplanets, pre-processing might not be necessary.

  5. How do you expect (at this early stage) that your visualisation application will load the dataset? E.g. database, from file, hard-coded in your javascript program, ... Why?

    6. In this project, I’d like to load the dataset from the file. Even the database can introduce more conveniences to access to data, the maintenance job of the database will increase the complexity of this project and it is not worth to do in this little project.

  3. What questions will the visualisation help your audience (not yourself!) to answer?

One of them is the relationship between time and attributes of exoplanets, like how many exoplanets were found in every year, the average radius of exoplanet found in every year, etc. The second one is the statistic value of exoplanet, like the exoplanets’ mass distribution, the comparison of the exoplanet number found in different method, etc. The third one is the combine of the above to, revealing the relationship between the statistic value of some specified attributes and the whole time range or a given time range.

    1. a visualisation that contains only one view/representation, but this representation is relatively novel and creative (not your standard bar charts etc);
